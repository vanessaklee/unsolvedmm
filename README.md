# Unsolvedmm

To start your Phoenix server:

  * Install dependencies with `mix deps.get`
  * Install Node.js dependencies with `cd assets && npm install`
  * Start Phoenix endpoint with `mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

## LiveView

Follow install instructions https://github.com/phoenixframework/phoenix_live_view#installation

Don't forget to install the new npm dependency.

  > npm install --prefix assets

If you had previously installed phoenix_live_view and want to get the latest javascript, then force an > install.

  > (cd assets && npm install --force phoenix_live_view)
